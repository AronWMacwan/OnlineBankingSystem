# Online Banking System

A **Secure Online Banking System** built in **Java** with a modern GUI. This project demonstrates real-world banking operations including account creation, transactions, fund transfers, balance checking, transaction history, and account deletion.

## Features
- ✅ Create Account with unique ID
- ✅ Login Authentication with Password
- ✅ Deposit and Withdraw Funds
- ✅ Transfer Money between Accounts
- ✅ Check Account Balance
- ✅ View Transaction History
- ✅ Delete Account
- ✅ Persistent Storage using Java Serialization
- ✅ Modern GUI using Java Swing
